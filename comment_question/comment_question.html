最常見的5個問題

  1. 如果我無法自己運送家具怎麼辦?

    2. IKEA大部份的商品都是平整包裝，你可以在商品頁面找到包裝的尺寸大小、重量等相關資訊，方便你在購買後自己搬回家，同時省下更多費用。

    IKEA大部份的商品都是平整包裝，你可以在商品頁面找到包裝的尺寸大小、重量等相關資訊，方便你在購買後自己搬回家，同時省下更多費用。 點選這裡 查詢運送服務的相關詳情。

  2. 我住的地方離IKEA很遠，可以透過傳真訂購IKEA的商品嗎?

    3. 我們的各分店有傳真或E-mail訂購服務可以接受訂購。
    首先 點選這裡找到離你最近的IKEA分店，下載傳真或E-mail訂購單。
    接著瀏覽本網站，將你想購買的商品資訊填入傳真或E-mail訂購單中。
    最後將訂購單傳真或 E-mail 到你選擇的IKEA分店，我們專業服務人員會和你聯絡。
    進一步洽詢各分店的傳真訂購/E-mail訂購資訊，請點選這裡與各分店連繫。

  3. 如何知道我想購買的商品在 IKEA店內有沒有貨?

    4. 網站與IKEA Store App上除了有數千種不同設計和價格的IKEA家具家飾商品供你瀏覽。在點入商品的詳細介紹頁面後，你可以使用查詢貨況功能確認想購買商品之貨況。
    也可以 點選這裡直接輸入你想購買的貨號或產品名稱，確認分店是否有貨。
    不過因為貨況可能隨時變動，建議你來店之前，來電做最後確認。歡迎 點選這裡確認各分店連絡方式。

  4. 我購買IKEA家具後，IKEA可以提供組裝服務嗎?

    5. 我們的商品除了包裝平整，方便你購買後直接帶走，還有完整的組裝說明，讓你按步驟組裝。
    當然，你也可以選擇我們專業的組裝服務，為你節省時間。組裝費用為產品售價的 6.5%，其費用未達$400者，最低以$400計算 (廚具、洗手台及窗簾訂製施工費另計)。
    歡迎 點選這裡查詢組裝服務的相關詳情。

  5. 我購買的商品需要退換貨，該怎麼做呢?

    6. 在IKEA，如果你對於購買的產品不滿意，你可在180天內(宜家卡卡友可在365天內)，攜帶完好的商品、原始包裝、統一發票及明細(刷卡購買者需信用卡與簽單)到原購買分店退換貨。
    完整退換貨資訊請 點選這裡。 或是你也可以與原購買分店客服連繫確認，請 點選這裡確認各分店連絡方式。
